# Figma-Ready Audit Exports

This folder contains second-format exports of the 17 audit text files.

## Best Option

Use the SVG files in `svg/`.

You can drag the SVG files into Figma or use Figma's import option. Each audit is laid out as a clean white artboard-style SVG with text elements.

## Paste Option

Use the files in `text/` if you want cleaner copy/paste text without Markdown symbols.

There is also one combined file:

`all-audits-figma-copy.txt`

## Combined Board

`all-audits-figma-board.svg` contains all audits stacked in one large SVG.

If Figma struggles with the combined board, import the individual SVG files instead.
