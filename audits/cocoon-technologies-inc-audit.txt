## Audit Summary

Cocoon Technologies Inc.'s website was reviewed from a public-facing perspective.

Main opportunities:

1. Improve speed and reduce what loads before the first view
2. Make the message, hierarchy, and CTA path easier to scan
3. Clean up code and third-party scripts that are not needed right away
4. Optimise images, video, and other visual assets
5. Improve search structure, page titles, and heading quality
6. Clean up accessibility and public-facing quality details
